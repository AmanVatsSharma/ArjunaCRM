### Coding a feature

從尋找適合初學者的任務開始您的旅程：

1. Navigate to the **[Issues](https://github.com/vedpragyabharat/arjuna/issues)** tab on the ArjunaCRM repository.
2. Filter by **[Good First Issue](https://github.com/vedpragyabharat/arjuna/labels/good%20first%20issue)** label to find tasks suited for newcomers.
3. 選擇一個問題，分叉存儲庫，開始貢獻。

<img src="/images/user-guide/github/good-first-issues.png" style={{width:'100%'}}/>

確保您被分配到這個問題，以避免與其他貢獻者的工作重疊。

### 行為守則

記住在整個貢獻過程中遵守 ArjunaCRM 的[行為守則](https://github.com/vedpragyabharat/arjuna/blob/main/.github/CODE_OF_CONDUCT.md)。
