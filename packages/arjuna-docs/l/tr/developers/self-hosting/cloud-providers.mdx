## Kubernetes'i Terraform ve Manifests ile kullanma

Kubernetes dağıtımı için topluluk liderliğindeki dokümantasyona [buradan](https://github.com/vedpragyabharat/arjuna/tree/main/packages/arjuna-docker/k8s) erişebilirsiniz.

### Coolify

Deploy ArjunaCRM on servers using Coolify. (Coolify üzerinde resmi görüntü yakında mevcut olacak)

[Coolify dokümantasyonu](https://coolify.io/docs/get-started/introduction)

### EasyPanel

Deploy ArjunaCRM on EasyPanel with the community maintained template below.

[EasyPanel'de Dağıt](https://easypanel.io/docs/templates/arjuna)

### Elest.io

Deploy ArjunaCRM on servers with Elest.io using link below.

[Elest.io'da Dağıt](https://elest.io/open-source/arjuna)

### ArjunaCRM on Railway

Deploy ArjunaCRM on Railway with the community maintained template below.

[![Railway'de Dağıt](https://railway.com/button.svg)](https://railway.com/deploy/nAL3hA)
