## About

Репозиторий ArjunaCRM на GitHub хранит множество ресурсов, таких как исходный код, документация, обсуждения и отслеживание проблем. Здесь вы сможете получить доступ ко всему коду, стоящему за ArjunaCRM.

[Посетите ArjunaCRM на GitHub](https://github.com/vedpragyabharat/arjuna)

## Вклад

Внесение вклада в проект ArjunaCRM на GitHub - это полезный способ помочь улучшить используемое вами программное обеспечение. Независимо от того, исправляете ли вы ошибки, предлагаете функции или улучшаете документацию, ваш вклад приветствуется.

### Сообщение об ошибках

Возникла проблема? [Create an issue](https://github.com/vedpragyabharat/arjuna/issues/new) on GitHub, providing as much detail as possible.

<img src="/images/user-guide/github/new-issue.png" style={{width:'100%'}}/>

### Предложения функций

Какие улучшения вы хотели бы видеть в ArjunaCRM? Независимо от уровня ваших технических знаний, вы можете присоединиться к [разговору здесь](https://github.com/vedpragyabharat/arjuna/discussions).

<img src="/images/user-guide/github/github-conversations.png" style={{width:'100%'}}/>

### Разработка функции

Начните свой путь с поиска задач для начинающих:

1. Перейдите на вкладку **[Issues](https://github.com/vedpragyabharat/arjuna/issues)** в репозитории ArjunaCRM.
2. Отфильтруйте по метке **[Good First Issue](https://github.com/vedpragyabharat/arjuna/labels/good%20first%20issue)**, чтобы найти задачи, подходящие для новичков.
3. Выберите задачу, форкните репозиторий и начните вносить вклад.

<img src="/images/user-guide/github/good-first-issues.png" style={{width:'100%'}}/>

Убедитесь, что вы назначены на проблему, чтобы избежать пересечения работы с другими участниками.

### Кодекс поведения

Помните, что нужно придерживаться [Кодекса поведения](https://github.com/vedpragyabharat/arjuna/blob/main/.github/CODE_OF_CONDUCT.md) ArjunaCRM в процессе вашего вклада.
