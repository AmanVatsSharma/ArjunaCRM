Дозволяє користувачам вибирати значення зі списку заданих варіантів.

<Tabs>
<Tab title="Usage">

```jsx
import { RecoilRoot } from 'recoil';
import { IconArjunaCRMStar } from 'arjuna-ui/display';

import { Select } from '@/ui/input/components/Select';

export const MyComponent = () => {

  return (
    <RecoilRoot>
      <Select
        className
        disabled={false}
        label="Select an option"
        options={[
          { value: 'option1', label: 'Option A', Icon: IconArjunaCRMStar },
          { value: 'option2', label: 'Option B', Icon: IconArjunaCRMStar },
        ]}
        value="option1"
      />
    </RecoilRoot>
  );
};

```

</Tab>
<Tab title="Props">

| Властивості | Тип         | Опис                                                                                                                                                                                                                                                   |
| ----------- | ----------- | ------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------ |
| className   | рядок       | Опційний клас CSS для додаткового стилювання                                                                                                                                                                                                           |
| відключено  | булевий тип | When set to `true`, disables user interaction with the component                                                                                                                                                                                       |
| label       | рядок       | The label to describe the purpose of the `Select` component                                                                                                                                                                                            |
| onChange    | функція     | Функція, яка викликається при зміні вибраних значень                                                                                                                                                                                                   |
| опції       | масив       | Представляє опції, доступні для компонента `Selected`. It's an array of objects where each object has a `value` (the unique identifier), `label` (the unique identifier), and an optional `Icon` |
| значення    | рядок       | Представляє поточно вибране значення. It should match one of the `value` properties in the `options` array                                                                                                                             |

</Tab>
</Tabs>
