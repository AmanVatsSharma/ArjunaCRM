## Kubernetes via Terraform og Manifests

Community-led documentation for Kubernetes deployment is available [here](https://github.com/vedpragyabharat/arjuna/tree/main/packages/arjuna-docker/k8s)

### Coolify

Distribuer ArjunaCRM på servere ved hjelp av Coolify. (offisielt bilde på Coolify vil snart være tilgjengelig)

[Coolify documentation](https://coolify.io/docs/get-started/introduction)

### EasyPanel

Distribuer ArjunaCRM på EasyPanel med malen vedlikeholdt av fellesskapet nedenfor.

[Distribuer på EasyPanel](https://easypanel.io/docs/templates/arjuna)

### Elest.io

Distribuer ArjunaCRM på servere med Elest.io ved hjelp av linken nedenfor.

[Distribuer på Elest.io](https://elest.io/open-source/arjuna)

### ArjunaCRM på Railway

Distribuer ArjunaCRM på Railway med malen vedlikeholdt av fellesskapet nedenfor.

[![Distribuer på Railway](https://railway.com/button.svg)](https://railway.com/deploy/nAL3hA)
