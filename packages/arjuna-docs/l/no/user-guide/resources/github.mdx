## Om

ArjunaCRM GitHub-repositoriet inneholder et bredt utvalg av ressurser som kildekode, dokumentasjon, diskusjoner og problemsporing. Dette er hvor du vil kunne få tilgang til den fullstendige koden bak ArjunaCRM.

[Besøk ArjunaCRM på GitHub](https://github.com/vedpragyabharat/arjuna)

## Bidra

Å bidra til ArjunaCRM-prosjektet på GitHub er en givende måte å hjelpe til med å forbedre programvaren du bruker. Enten du fikser feil, foreslår funksjoner eller forbedrer dokumentasjon, er dine bidrag velkomne.

### Reporting Issues

Møtt på et problem? [Opprett et problem](https://github.com/vedpragyabharat/arjuna/issues/new) på GitHub og gi så mye informasjon som mulig.

<img src="/images/user-guide/github/new-issue.png" style={{width:'100%'}}/>

### Foreslå funksjoner

Hvilke forbedringer vil du se på ArjunaCRM? Uansett ditt tekniske kunnskapsnivå, kan du delta i [diskusjonen her](https://github.com/vedpragyabharat/arjuna/discussions).

<img src="/images/user-guide/github/github-conversations.png" style={{width:'100%'}}/>

### Coding a feature

Start din reise ved å finne nybegynnervennlige oppgaver:

1. Naviger til **[Issues](https://github.com/vedpragyabharat/arjuna/issues)**-fanen på ArjunaCRM-repositoriet.
2. Filtrer etter **[Good First Issue](https://github.com/vedpragyabharat/arjuna/labels/good%20first%20issue)**-etikett for å finne oppgaver for nykommere.
3. Pick an issue, fork the repository, and start contributing.

<img src="/images/user-guide/github/good-first-issues.png" style={{width:'100%'}}/>

Sørg for at du er tildelt saken for å unngå overlappende arbeid med andre bidragsytere.

### Code of Conduct

Husk å følge ArjunaCRM's [Code of Conduct](https://github.com/vedpragyabharat/arjuna/blob/main/.github/CODE_OF_CONDUCT.md) gjennom hele din bidragsprosess.
