## О томе

GitHub репозиторијум за ArjunaCRM садржи велику разноликост ресурса као што су изворни код, документација, дискусије и праћење проблема. Овде ћете моћи да приступите целокупном коду иза ArjunaCRM.

[Посетите ArjunaCRM на GitHub-у](https://github.com/vedpragyabharat/arjuna)

## Доприношење

Contributing to the ArjunaCRM project on GitHub is a rewarding way to help improve the software you use. Без обзира на то да ли поправљате грешке, предлажете функције или побољшавате документацију, ваши доприноси су добродошли.

### Пријављивање проблема

Наишли сте на проблем? [Креирајте проблем](https://github.com/vedpragyabharat/arjuna/issues/new) на GitHub-у, уз што више детаља.

<img src="/images/user-guide/github/new-issue.png" style={{width:'100%'}}/>

### Предлагање функција

Која побољшања бисте желели видети на ArjunaCRM? Без обзира на вашу стручност, можете се придружити [разговору овде](https://github.com/vedpragyabharat/arjuna/discussions).

<img src="/images/user-guide/github/github-conversations.png" style={{width:'100%'}}/>

### Кодирање функције

Започните своје путовање проналажењем задатака погодних за почетнике:

1. Навигирајте на **[Проблеми](https://github.com/vedpragyabharat/arjuna/issues)** таб у ArjunaCRM репозиторијуму.
2. Филтрирајте према ознаци **[Добар први проблем](https://github.com/vedpragyabharat/arjuna/labels/good%20first%20issue)** за проналажење задатака погодних за новајлије.
3. Одаберите проблем, форкујте репозиторијум и почните да доприносите.

<img src="/images/user-guide/github/good-first-issues.png" style={{width:'100%'}}/>

Уверите се да сте додељени проблему да бисте избегли преклапање рада са другим доприносима.

### Правила понашања

Запамтите да се придржавате [правила понашања](https://github.com/vedpragyabharat/arjuna/blob/main/.github/CODE_OF_CONDUCT.md) ArjunaCRM током целог процеса доприноса.
