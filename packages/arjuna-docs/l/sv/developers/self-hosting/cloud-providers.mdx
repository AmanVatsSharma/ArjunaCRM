Gemenskapsledd dokumentation för Kubernetes-implementering är tillgänglig [här](https://github.com/vedpragyabharat/arjuna/tree/main/packages/arjuna-docker/k8s)

Distribuera ArjunaCRM på servrar med Coolify. (officiell bild på Coolify kommer snart att finnas tillgänglig)

[Coolify documentation](https://coolify.io/docs/get-started/introduction)

[Distribuera på EasyPanel](https://easypanel.io/docs/templates/arjuna)

[Distribuera på Elest.io](https://elest.io/open-source/arjuna)
