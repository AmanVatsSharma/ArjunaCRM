## Steg 1: Registrering

1. Navigera till [ArjunaCRM Sign Up](https://app.vedpragya.com).
2. Välj din föredragna registreringsmetod:
   - **Fortsätt med Google** för Google-kontoregistrering.
   - **Fortsätt med Microsoft** för Microsoft-kontoregistrering.
   - Eller, **Fortsätt med E-post** för e-postregistrering.

<VimeoEmbed videoId="927066829" title="Video demonstration" />
## Step 2: Choosing a Trial Period

Välj mellan två provperioder:

### 30 dagar

Med kreditkort

### 7 dagar

Utan kreditkort

Båda provperioderna inkluderar:

- Full tillgång
- Obegränsat med kontakter
- E-post integration
- Anpassade objekt
- API & Webhooks

Du kan klicka på "Ändra plan" för att välja en annan plan eller faktureringsintervall.

<img src="/images/user-guide/create-workspace/choose-plan.png" style={{width:'100%'}}/>

## Steg 3: Betalningsbekräftelse & Kontoinställning

Efter betalningsgodkännande via Stripe, dirigeras du till att skapa ditt arbetsutrymme och din användarprofil. Kom ihåg att du kan avbryta din prenumeration när som helst.
