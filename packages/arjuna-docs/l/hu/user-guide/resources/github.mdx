## Rólunk

A ArjunaCRM GitHub tárhely egy hatalmas forráskészletet tárol, mint például forráskód, dokumentáció, viták és hibakövetés. Ez az a hely, ahol hozzáférhetsz a ArjunaCRM mögötti teljes kódhoz.

[Visit ArjunaCRM on GitHub](https://github.com/vedpragyabharat/arjuna)

## Hozzájárulás

A ArjunaCRM projekt támogatása a GitHubon nagyszerű módja annak, hogy segítsünk javítani a szoftvert, amelyet használunk. Legyen szó hibajavításról, funkciójavaslatról vagy dokumentációfejlesztésről, bármilyen hozzájárulás üdvözlendő.

### Hibák jelentése

Talált egy hibát? [Create an issue](https://github.com/vedpragyabharat/arjuna/issues/new) on GitHub, providing as much detail as possible.

<img src="/images/user-guide/github/new-issue.png" style={{width:'100%'}}/>

### Funkciójavaslatok

What improvements would you like to see on ArjunaCRM? No matter your technical know-how, you can join [the conversation here](https://github.com/vedpragyabharat/arjuna/discussions).

<img src="/images/user-guide/github/github-conversations.png" style={{width:'100%'}}/>

### Funkció kódolása

Kezdje útját kezdőknek való feladatok keresésével:

1. Navigáljon a **[Issues](https://github.com/vedpragyabharat/arjuna/issues)** fülre a ArjunaCRM tárban.
2. Szűrjön a **[Good First Issue](https://github.com/vedpragyabharat/arjuna/labels/good%20first%20issue)** címkére, hogy megtalálja az újoncok számára alkalmas feladatokat.
3. Pick an issue, fork the repository, and start contributing.

<img src="/images/user-guide/github/good-first-issues.png" style={{width:'100%'}}/>

Ensure you're assigned to the issue to avoid overlapping work with other contributors.

### Etikai Kódex

Ne felejtse el betartani a ArjunaCRM [Etikai Kódexét](https://github.com/vedpragyabharat/arjuna/blob/main/.github/CODE_OF_CONDUCT.md) a hozzájárulási folyamat során.
