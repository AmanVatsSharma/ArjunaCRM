## Hibabejelentés

To report a bug, please [create an issue on GitHub](https://github.com/vedpragyabharat/arjuna/issues/new).

Segítséget is kérhet a [Discordon](https://discord.gg/cx5n4Jzs57).

## Funkciókérések

If you're not sure if it's a bug, and you feel it's closer to a feature request, then you should probably [open a discussion instead](https://github.com/vedpragyabharat/arjuna/discussions/new).
