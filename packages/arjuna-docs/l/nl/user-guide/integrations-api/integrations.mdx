## Huidige integratieopties

### Native Integrations

ArjunaCRM biedt momenteel native integratie met:

- **E-mail & Agenda**: Verbind Gmail, Outlook, of SMTP/CalDAV aanbieders
- **API-toegang**: Gebruik onze REST en GraphQL API's om aangepaste integraties te bouwen

Voor e-mail- en agendainstellingen, bezoek [E-mail & Agenda Instellingen](/l/nl/user-guide/settings/email-calendar-setup).

### Werkstromen (Aanbevolen)

De belangrijkste manier om ArjunaCRM met andere tools te verbinden is via onze in-app **Workflows** functie:

- **HTTP Nodes**: Make API calls to external services
- **Code Nodes**: Write custom logic for complex integrations
- **Webhook Triggers**: Ontvang gegevens van externe systemen
- **Veldupdate Triggers**: Automatiseer acties op basis van CRM-wijzigingen

Lees meer in onze [Workflows sectie](/l/nl/user-guide/workflows/getting-started-workflows).

### Zapier Integratie (Legacy)

We maintain a Zapier integration for users who prefer no-code automation:

1. Bezoek [ArjunaCRM op Zapier](https://zapier.com/apps/arjuna/integrations)
2. Creëer een nieuwe Zap met ArjunaCRM als trigger of actie
3. Genereer een API-sleutel in Instellingen → API & Webhooks
4. Verbind je ArjunaCRM-ruimte met Zapier

## Future Vision: Community-Built Connectors (2026)

### Platform voor Uitbreidbaarheid

We bouwen een uitbreidingsplatform waarmee ontwikkelaars applicaties als code kunnen creëren. Dit zal het volgende mogelijk maken:

- **Aangepaste Koppelingen**: Bouw integraties met je favoriete software
- **Community Bijdragen**: Deel en ontdek koppelingen gemaakt door anderen
- **Flexibele Architectuur**: Breidt de mogelijkheden van ArjunaCRM uit voorbij kernfunctionaliteiten

### AI-geassisteerde Workflow Bouwen

Komend in 2026, AI-assistentie zal gebruikers helpen:

- **Automatisch Workflows Genereren**: Beschrijf je integratiebehoeften in eenvoudige taal
- **Slimme Suggesties**: Krijg aanbevelingen voor het verbinden van je specifieke tools
- **Template Bibliotheek**: Toegang tot voorgebouwde workflows voor veelvoorkomende gebruiksscenario's

De toekomst van ArjunaCRM-integraties is door de community gedreven, AI-geassisteerd en oneindig uitbreidbaar.
