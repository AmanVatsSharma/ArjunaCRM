## Over

De ArjunaCRM GitHub-repository host een breed scala aan bronnen zoals broncode, documentatie, discussies en probleemopvolging. Hier kunt u de volledige code achter ArjunaCRM openen.

[Bezoek ArjunaCRM op GitHub](https://github.com/vedpragyabharat/arjuna)

## Bijdragen

Bijdragen aan het ArjunaCRM-project op GitHub is een lonende manier om de software die u gebruikt te verbeteren. Of je nu bugs repareert, functies voorstelt, of documentatie verbetert, je bijdragen zijn welkom.

### Problemen melden

Heeft u een probleem? [Maak een probleem aan](https://github.com/vedpragyabharat/arjuna/issues/new) op GitHub en geef zoveel mogelijk details.

<img src="/images/user-guide/github/new-issue.png" style={{width:'100%'}}/>

### Functies voorstellen

Welke verbeteringen zou je graag zien bij ArjunaCRM? Ongeacht je technische ervaring, je kunt [hier deelnemen aan het gesprek](https://github.com/vedpragyabharat/arjuna/discussions).

<img src="/images/user-guide/github/github-conversations.png" style={{width:'100%'}}/>

### Een functie coderen

Begin je reis met het vinden van taken voor beginners:

1. Navigeer naar het **[Issues](https://github.com/vedpragyabharat/arjuna/issues)** tabblad op de ArjunaCRM repository.
2. Filter op het label **[Good First Issue](https://github.com/vedpragyabharat/arjuna/labels/good%20first%20issue)** om taken te vinden die geschikt zijn voor beginners.
3. Kies een issue, fork de repository, en begin met bijdragen.

<img src="/images/user-guide/github/good-first-issues.png" style={{width:'100%'}}/>

Zorg ervoor dat je aan het issue bent toegewezen om te voorkomen dat je werk overlapt met andere bijdragers.

### Gedragscode

Vergeet niet om je te houden aan de [Gedragscode](https://github.com/vedpragyabharat/arjuna/blob/main/.github/CODE_OF_CONDUCT.md) van ArjunaCRM gedurende je bijdrageproces.
