## Toteutuspalvelut

Tarvitsetpa apua aloittamiseen tai kehittyneempien mukautusten luomiseen, meillä on ratkaisu.

### Perehdytyspaketit

Saat apua ydintiimiltämme ArjunaCRMn työtilan määrittämiseen 4-tunnin [Perehdytyspaketeilla](https://vedpragya.com/onboarding-packages):

- **Tietomallin suunnittelu**: Suunnittele ja luo mukautettu tietomalli objekteilla, kentillä ja suhteilla
- **Datan siirto**: Siirrä olemassa olevat tietosi nykyisestä CRM:stäsi ArjunaCRMyn
- **Työnkulkujen luominen**: Luo mukautettuja työnkulkuja tukemaan liiketoimintaprosessejasi

### Toteutuskumppanit

Tee yhteistyötä sertifioitujen ArjunaCRMn kumppanien kanssa edistyneempien mukautusten ja integrointien toteuttamiseksi. Ota yhteyttä tiimiimme osoitteessa contact@vedpragya.com löytääksesi sopivat [kumppanit](https://vedpragya.com/implementation-services).
