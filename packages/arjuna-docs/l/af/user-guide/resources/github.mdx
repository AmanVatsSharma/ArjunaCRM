## Omtrent

Die ArjunaCRM GitHub-bewaarplek huisves 'n wye verskeidenheid hulpbronne soos bronkode, dokumentasie, besprekings en kwessieopsporing. Hier is waar jy die volledige kode agter ArjunaCRM kan bekom.

[Besoek ArjunaCRM op GitHub](https://github.com/vedpragyabharat/arjuna)

## Bydra

Contributing to the ArjunaCRM project on GitHub is a rewarding way to help improve the software you use. Of jy nou foute regstel, funksies voorstel, of dokumentasie verbeter, jou bydraes is welkom.

### Reporting Issues

Encounter an issue? [Skep 'n probleem](https://github.com/vedpragyabharat/arjuna/issues/new) op GitHub en verskaf soveel detail as moontlik.

<img src="/images/user-guide/github/new-issue.png" style={{width:'100%'}}/>

### Suggesting Features

Watter verbeterings wil jy graag op ArjunaCRM sien? Maak nie saak jou tegniese kennis nie, jy kan [aan die gesprek deelneem](https://github.com/vedpragyabharat/arjuna/discussions).

<img src="/images/user-guide/github/github-conversations.png" style={{width:'100%'}}/>

### Kodeer 'n kenmerk

Begin jou reis deur beginnervriendelike take te vind:

1. Navigeer na die **[Issues](https://github.com/vedpragyabharat/arjuna/issues)** bladsy van die ArjunaCRM-bewaarplek.
2. Filter deur **[Good First Issue](https://github.com/vedpragyabharat/arjuna/labels/good%20first%20issue)** etiket om take vir beginners te vind.
3. Kies 'n probleem, kloon die bewaarplek en begin bydra.

<img src="/images/user-guide/github/good-first-issues.png" style={{width:'100%'}}/>

Maak seker dat jy aan die probleem toegewys is om oorvleuelende werk met ander bydraers te vermy.

### Gedragskode

Onthou om deurentyd aan ArjunaCRM se [gedragskode](https://github.com/vedpragyabharat/arjuna/blob/main/.github/CODE_OF_CONDUCT.md) te voldoen.
