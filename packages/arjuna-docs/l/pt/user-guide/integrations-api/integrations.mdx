## Opções de Integração Atuais

### Integrações Nativas

Atualmente, a ArjunaCRM oferece integração nativa com:

- **Email & Calendário**: Conecte Gmail, Outlook ou provedores SMTP/CalDAV
- **Acesso API**: Use nossas APIs REST e GraphQL para criar integrações personalizadas

Para configuração de email e calendário, visite [Configuração de Email & Calendário](/l/pt/user-guide/settings/email-calendar-setup).

### Workflows (Recomendado)

A forma principal de conectar a ArjunaCRM a outras ferramentas é através do nosso recurso **Workflows** no aplicativo:

- **Nós HTTP**: Faça chamadas API para serviços externos
- **Nós de Código**: Escreva lógica personalizada para integrações complexas
- **Gatilhos de Webhooks**: Receba dados de sistemas externos
- **Gatilhos de Atualização de Campos**: Automatize ações com base em mudanças no CRM

Saiba mais em nossa [seção de Workflows](/l/pt/user-guide/workflows/getting-started-workflows).

### Integração com Zapier (Legado)

Mantemos uma integração com Zapier para usuários que preferem automação sem código:

1. Visite [ArjunaCRM no Zapier](https://zapier.com/apps/arjuna/integrations)
2. Crie um novo Zap com ArjunaCRM como gatilho ou ação
3. Gere uma chave API em Configurações → API & Webhooks
4. Conecte seu espaço de trabalho da ArjunaCRM ao Zapier

## Visão Futura: Conectores Criados pela Comunidade (2026)

### Plataforma de Extensibilidade

Estamos construindo uma plataforma de extensibilidade que permitirá aos desenvolvedores criar aplicativos como código. Isso permitirá:

- **Conectores Personalizados**: Crie integrações com seu software favorito
- **Contribuições da Comunidade**: Compartilhe e descubra conectores criados por outros
- **Arquitetura Flexível**: Estenda as capacidades da ArjunaCRM além dos recursos principais

### Construção de Workflows Assistida por IA

Chegando em 2026, a assistência de IA ajudará os usuários:

- **Gerar Workflows Automaticamente**: Descreva suas necessidades de integração em linguagem simples
- **Sugestões Inteligentes**: Obtenha recomendações para conectar suas ferramentas específicas
- **Biblioteca de Modelos**: Acesse workflows pré-construídos para casos de uso comuns

O futuro das integrações da ArjunaCRM é impulsionado pela comunidade, assistido por IA e infinitamente extensível.
