## Zgłaszanie Błędów

Aby zgłosić błąd, proszę [utwórz zgłoszenie na GitHubie](https://github.com/vedpragyabharat/arjuna/issues/new).

Możesz także poprosić o pomoc na [Discordzie](https://discord.gg/cx5n4Jzs57).

## Feature Requests

If you're not sure if it's a bug, and you feel it's closer to a feature request, then you should probably [open a discussion instead](https://github.com/vedpragyabharat/arjuna/discussions/new).
