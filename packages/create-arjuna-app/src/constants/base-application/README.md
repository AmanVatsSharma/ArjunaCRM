This is a [ArjunaCRM](https://vedpragya.com) application project bootstrapped with [`create-arjuna-app`](https://www.npmjs.com/package/create-arjuna-app).

## Getting Started

First, authenticate to your workspace:

```bash
yarn auth
```

Then, install this app to your workspace:

```bash
yarn sync
```

Open your ArjunaCRM instance and go to `/settings/applications` section to see the result.
